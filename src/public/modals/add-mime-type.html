📄 Add MIME Type

Enter a valid MIME type (e.g., image/png, video/mp4, application/json)
⚠️ Security Warning
Adding certain MIME types can be a security risk:
  • image/svg+xml - SVG files can contain JavaScript, enabling XSS attacks
  • text/html - HTML files can execute scripts in browsers
  • application/x-* - Executable or script formats may run malicious code
  • application/javascript - JavaScript files can be executed by browsers
Only add MIME types for file formats you explicitly need and trust.
ℹ️ Finding MIME Types
To find the correct MIME type for a file extension:
  • Check IANA's official registry
  • Use file --mime-type filename.ext on Linux/Mac
  • Search "[extension] MIME type" (e.g., ".pdf MIME type" → "application/pdf")
